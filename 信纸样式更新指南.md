# 女书密信 - 自定义信纸及样式更新指南

当前项目已内置了三种**信纸花纹**，同时也支持用户**在页面直接上传自己的背景图**。如果您后续希望更新代码，向系统中内置新的可选主题，请参考以下指南。

## 1. 信纸素材尺寸要求

由于移动端和 PC 端保存图片的生成规则是建立在固定的基础画板大小上，如果想确保没有任何裁剪变形，素材建议按以下标准准备：

| 样式类型 | 尺寸要求 | 格式建议 | 说明 |
| --- | --- | --- | --- |
| **四周角花** (如默认花纹) | 140px × 140px | `.png` (纯透明底色) | 只需要准备一张图片，代码会自动翻转并应用到四个角。 |
| **全幅边框** (如图 1 青萝) | 760px × 1080px (或比例为 19:27 的大小) | `.png` (中心透明) | 直接铺满整个信纸，透过中间的透明区域显示底色。 |
| **全幅背景** (如图 2 古风) | 760px × 1080px (宽高比约 0.7) | `.jpg` 或 `.png` | 画板的实际渲染大小是 380×540 (为了高清，导出时放大了 2 倍即 760×1080)。图片会按 `cover` 方式缩放铺满。 |

> **一句话总结**：想更换全幅画卷，请制作 **760×1080 像素** 的图片以达到最高清、最完美的适配效果。

## 2. 如何内置新的信纸样式

如果您想把某个新的好看背景固定下来，让大家可以在下拉框选择，请按以下三步进行更新：

### 第一步：将文件放入 `images` 文件夹
把准备好的图片放到 `images` 文件夹下，例如 `images/bg-style3.jpg`。

### 第二步：修改页面下拉菜单 (`index.html`)
找到 `<select id="theme-selector">`，在 `custom` 前面添加一个新的选项：
```html
<option value="style3">我的新信纸</option>
```

### 第三步：在 CSS 样式中添加它 (`css/style.css`)
找到与 `#card-bg-style2` 相关的地方，复制一份对应新的 id：
```css
/* 新加一组 */
#card-front.theme-style3 { background-color: transparent; }

#card-bg-style3 {
  position: absolute;
  inset: 0;
  width: 100%;
  height: 100%;
  background-size: cover;
  background-position: center;
  background-image: url('../images/bg-style3.jpg'); /* 你的图片名字 */
  z-index: 1;
}
```
并在 HTML 里的 `<div id="card-front">` 内部，增加对应的 div 层：
```html
<div id="card-bg-style3" class="hidden"></div>
```

### 第四步：在 JavaScript 中应用逻辑和保存截图 (`js/letter.js`)

**1. 界面呈现：**
找到 `function applyTheme(theme)` 方法，把新的 div 加进去：
```javascript
const bgStyle3 = document.getElementById('card-bg-style3'); // 顶部获取元素
// ...
bgStyle3.classList.add('hidden'); // 加入隐藏组
// ...
else if (theme === 'style3') {
    document.getElementById('card-front').style.backgroundColor = 'transparent';
    bgStyle3.classList.remove('hidden');
}
```

**2. 截图保存：**
找到 `btnSave.addEventListener` 下面的 Canvas 绘制代码：
```javascript
if (theme === 'style2' || theme === 'style3' || theme === 'custom') {
    // 增加逻辑判断
    let bgImgSrc = customImageSrc;
    if (theme === 'style2') bgImgSrc = 'images/bg-style2.jpg';
    if (theme === 'style3') bgImgSrc = 'images/bg-style3.jpg'; // 对应的图片路径
    
    // ... 后续代码会自动把它画出来
}
```

## 3. 面向用户的“上传自定义的信纸”功能
现在的页面上已经内置了**“上传自定义信纸...”**选项（位于主题下拉框的最底下）。
当用户选中此选项，系统会唤起手机或电脑的文件选择器，用户在相册里选中照片后，该照片就会**直接预览在卡片底层**，并不需要上传或改代码。
当用户点击“保存图片”时，新生成的密信也会完美带有该用户自己指定的相片壁纸。
